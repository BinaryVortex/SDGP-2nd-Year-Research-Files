# 📘 2nd Year SDGP Research Files

## 📌 Overview

This repository contains all research materials, documentation, and
supporting resources for the **Second Year Software Development Group
Project (SDGP)**.\
It serves as a centralized space to organize ideas, track progress, and
document findings throughout the research phase of the project.

------------------------------------------------------------------------

## 🎯 Objectives

-   Conduct in-depth research related to the SDGP project domain\
-   Analyze existing solutions and identify gaps\
-   Define problem statements and potential approaches\
-   Support the development phase with validated insights

------------------------------------------------------------------------

## 📂 Project Structure

/research-papers -\> Collected academic papers and references\
/literature-review -\> Summaries and analysis of research papers\
/ideas -\> Brainstorming notes and concept drafts\
/diagrams -\> Architecture, workflows, and system designs\
/reports -\> Formal documentation and submissions

------------------------------------------------------------------------

## 🔍 Research Areas

-   Problem Domain Analysis\
-   Existing System Evaluation\
-   Technology Stack Exploration\
-   Feasibility Study\
-   User Requirements & Use Cases

------------------------------------------------------------------------

## 🛠️ Tools & Technologies

-   📄 Documentation: Microsoft Word / Google Docs\
-   📊 Diagrams: Draw.io / Figma\
-   💻 Version Control: Git & GitHub\
-   🧠 Research Sources: Google Scholar, IEEE, ResearchGate

------------------------------------------------------------------------

## 🚀 How to Use
